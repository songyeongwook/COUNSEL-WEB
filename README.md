# 상담 관리 프로젝트

## 프로젝트 소개
JSP와 Servlet MVC 패턴을 기반으로 제작한 상담 관리 웹 프로젝트입니다.  
사용자와 상담사(관리자) 기능을 분리하여 상담 예약, 공지사항, 상담 이력, 피드백 파일 관리 등을 구현하였습니다.

주요 기능:
- 회원가입 / 로그인 / 로그아웃
- 아이디 및 비밀번호 찾기
- 상담 예약 및 상담사 지정
- 상담 이력 조회
- 공지사항 관리
- 상담 피드백 파일 다운로드
- 관리자(상담사) 전용 관리 페이지
- 예약 일정 확인 기능

프로젝트는 JSP + Servlet MVC 구조로 개발되었으며,
DB 연동 및 세션 기반 사용자 인증 기능을 구현하였습니다.

# 프로젝트 후기

프로젝트를 진행하면서 JSP와 Servlet 기반 MVC 구조를 실제로 적용하며
프론트엔드와 백엔드 연결 과정을 학습할 수 있었습니다.

또한 DB 연동, 세션 관리, 파일 처리, 예약 기능 등을 직접 구현하며
웹 프로젝트 구조와 서버 동작 원리를 이해할 수 있었습니다.
